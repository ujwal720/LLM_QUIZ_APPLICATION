# 📝 AI Quiz Evaluator: Full-Stack LLM Application

A full-stack AI application that parses study materials (PDFs), generates structured Multiple Choice Questions (MCQs) using the **Mistral-7B** Large Language Model, and evaluates user performance in real-time.

---

## 🚀 Features
* **PDF Parsing:** Extracts and chunks text from complex documents using `PyPDF` and `LangChain`.
* **LLM Intelligence:** Powered by **Mistral-7B-Instruct-v0.1**, quantized to 4-bit to run efficiently on T4 GPUs.
* **Structured Output:** Generates strictly formatted JSON data for seamless frontend integration.
* **Interactive UI:** Built with **Streamlit**, featuring a form-based quiz interface and scoring metrics.
* **Cloud-to-Local Tunneling:** Uses **Ngrok** to bridge a local frontend with high-performance cloud inference (Kaggle).

---

## 🏗️ Architecture
The project follows a distributed architecture to handle the compute-heavy requirements of Large Language Models:

1.  **Frontend (Local):** Streamlit app for file uploading and quiz interaction.
2.  **Tunnel (Ngrok):** Secure bridge connecting the local machine to the cloud backend.
3.  **Backend (Kaggle):** FastAPI server managing the model inference on a T4 GPU.
4.  **Model (Mistral-7B):** 4-bit quantized LLM generating the questions.



---

## 🛠️ Tech Stack
* **Language:** Python 3.10+
* **LLM Framework:** Hugging Face Transformers, BitsAndBytes (4-bit Quantization)
* **Backend:** FastAPI, Uvicorn
* **Frontend:** Streamlit
* **Data Handling:** LangChain (Recursive Text Splitting), PyPDF
* **Connectivity:** PyNgrok

## ⚙️ Setup Instructions

### 1. Backend (Kaggle/Cloud)
1.  Open the provided notebook in Kaggle.
2.  Enable **GPU T4 x2** accelerator and **Internet Access**.
3.  Add your `NGROK_TOKEN` to Kaggle Secrets.
4.  Run the cells to start the FastAPI server. Copy the generated `ngrok-free.dev` URL.

### 2. Frontend (Local)
1.  Clone this repository:
    ```bash
    git clone [https://github.com/yourusername/ai-quiz-evaluator.git](https://github.com/yourusername/ai-quiz-evaluator.git)
    cd ai-quiz-evaluator
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Create a `.env` file in the root directory:
    ```text
    NGROK_URL=[https://your-unique-id.ngrok-free.dev](https://your-unique-id.ngrok-free.dev)
    ```
4.  Run the application:
    ```bash
    streamlit run app.py
    ```

## 📝 Usage
1.  Upload a study PDF (e.g., Biology notes, Research papers).
2.  Click **"Generate Quiz"**.
3.  Answer the 3 MCQs generated by the AI.
4.  Click **"Submit Final Answers"** to see your score and the correct answers.

---

## 🛡️ Key Optimizations
* **Memory Efficiency:** Used `low_cpu_mem_usage` and `torch.float16` to prevent GPU kernel crashes.
* **Response Reliability:** Implemented Regex-based JSON extraction to handle LLM conversational filler.
* **UX Stability:** Utilized Streamlit `st.form` and `st.session_state` to prevent UI flickering during interaction.
